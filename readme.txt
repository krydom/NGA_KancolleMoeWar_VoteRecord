NGA舰萌计票辅助系统v5 by 江天万里霜
v7/8更新 by krydom

特性：
* 完全封装了整个计票过程，从打开网页到生成计票结果文件都在一个程序内实现。
* 自带自动识别功能，可以自动完成80%以上的工作，人工只需要查漏补缺。
* 纯键盘操作，计票速度极快。
* 支持保存进度，以及标记特定的楼层。
* 因为只花了一晚上来写，天知道会有什么bug。

使用方法：
详情见 教程.docx

moewar.exe是封装好的可执行文件，运行这个就可以了；Win10下可用，别的环境就有请沙雕群友们测试了。

moewar.py是源代码。zh_wiki.py和langconv.py是实现简繁转换的两个包。

config.ini是配置和存档文件。

vote.json是存放帖子内容的文件。

nickname_kantai.txt 和 nickname_teki.txt 分别是是舰娘/深海别名表，用来读取别名。

title.txt 和 starter.txt 保存了标题/楼主内容。

计票结果会输出为result.csv。

有什么不懂的或者要汇报bug直接群内@就好。
